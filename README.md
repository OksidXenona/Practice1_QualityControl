# Практика, 1 курс.
## Кейс 6. Контроль качества: поиск дефектов изделий по фото

### Контекст и цель:
Производство хочет отбраковывать изделия по фотографии на конвейере: царапины,
сколы, загрязнения, деформации. Цель: детектировать дефект (норма / брак) и
локализовать его на изображении.

### Задачи практики:
• обработка изображений изделий;
• обучить модель обнаружения аномалий на «нормальных» снимках;
• локализовать дефектную область (тепловая карта/маска);
• оценить качество.

### Постановка задачи и подготовка данных:
Тип задачи. Обнаружение аномалий на изображениях
Вход/выход. Вход: изображение. Выход: оценка аномальности изображения и карта
локализации дефекта.
